Hitman Peacock Powershell

This project is an automation utility designed to seamlessly download, deploy, and configure the Peacock project local server emulator for the Hitman trilogy. Instead of manually editing host files, verifying port mappings, or running several bat files, this Powershell setup handles everything in a single execution line.

Project Overview

The script automates dependency checks, fetches the latest build releases directly, and properly maps the connection certificates so your client game can interface with the custom backend server environment safely without interfering with your official network settings.

Compatibility Note

This automation framework is built and tested against official, legitimately purchased digital distributions of the Hitman trilogy. It assumes a legal, unmodified game installation and doesn't provide support for modified or illegitimately obtained copies.
