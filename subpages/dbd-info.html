DBD FPS Unlocker

Dead by Daylight has a native cap of 120 FPS. If you have a high-refresh-rate monitor (144Hz, 240Hz, etc.), the game won't go past 120 no matter how powerful your PC is.

This tool simply removes that limit. No bloat, no background processes — just a lightweight patch that lets the game run at whatever your hardware can push.

Just keep in mind — you need to set a limit your PC can actually handle. If your FPS drops below whatever you've set, the character starts running in slow-motion. Looks hilarious, but it completely ruins any chase.

Quick Notes

  • Works with Steam and Epic Games versions on Windows
  • No installation needed — just run the .exe
  • Requires administrator permissions so it can modify the engine config file without issues
  • Replaces the engine config file and sets it to read-only — game updates won't touch it
