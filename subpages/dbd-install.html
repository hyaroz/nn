Installation Guide

Setting up the Dead by Daylight FPS Unlocker is extremely straightforward and takes less than a minute. The utility automatically locates your game configuration and applies the patch, saving you the trouble of digging through hidden DBD directories.

What you need

  • Dead by Daylight on Steam or Epic Games
  • Windows 10 or 11
  • Game fully closed before patching

How to use

  • Step 1: Download the latest Dead by Daylight FPS Unlocker.exe from the download page.
  • Step 2: Right-click the .exe and select Run as administrator — the tool needs admin permissions to modify the engine config file.
  • Step 3: The tool automatically detects your Dead by Daylight installation and displays Status: Ready.
  • Step 4: Enter your desired frame rate in the Target FPS input box (e.g., 144, 165, or 240) and click the green Unlock FPS button.
  • Step 5: Launch the game and enjoy uncapped FPS. Disable V-Sync in the game settings if you want the full boost.

Reverting

Run the program again and click the red Revert changes button to restore the original config.

Note: The tool sets the engine config to read-only, so game updates won't override it. You only need to run this once.
