Hitman Peacock Powershell

Version 0.0.0 — currently in the early planning phase. Nothing to download or install yet, but here's what this project is all about.

What is this?

Hitman Peacock Powershell is an automation script that will handle the entire setup of the Peacock local server emulator for the Hitman trilogy (World of Assassination). No more manually editing host files, messing with port forwarding, or running a bunch of random .bat files — this script will do everything in one go.

What will it do?

  • Automatically check and install all required dependencies (Node.js, Git, certificates)
  • Download and set up the latest Peacock server build directly from the official repository
  • Configure the Windows host file and SSL certificates so the game talks to your local server
  • Launch the server with a single command — no manual configuration needed
  • Clean up everything if you want to revert back to the official servers

Planned Features

  • One-command setup and teardown
  • Automatic dependency detection and installation
  • Support for both Steam and Epic Games versions
  • Custom server directory selection
  • Safe rollback to original game settings

Compatibility Note

This script is being designed for legitimately owned copies of the Hitman trilogy. It assumes you have a legal game installation and does not support cracked or pirated versions.
