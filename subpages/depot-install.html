Installation Guide

DepotDowngrader ships as a standalone Windows utility — no complicated setup required. Follow the steps below to download a specific depot build for a game you already own on Steam.

Prerequisites

  • A legitimate copy of the target game in your Steam library.
  • Windows 10 or 11 (64-bit).
  • The official Steam client installed and logged in.
  • Enough free disk space for the depot version you want to download.

Step-by-Step Setup

  • Step 1: Download the latest release from the project's GitHub repository.
  • Step 2: Extract the archive to a folder of your choice (e.g., C:\Tools\DepotDowngrader).
  • Step 3: Launch DepotDowngrader.exe. On first run, sign in with your Steam credentials when prompted — the same account that owns the game.
  • Step 4: Enter the Steam App ID of the game, then pick the manifest (build) you want from the list or paste a known manifest ID directly.
  • Step 5: Choose the output directory and start the download. Once finished, point Steam to the downgraded files or replace the game folder as described in the release notes for your specific title.

Reminder: You can only download depots for games and DLCs already licensed on your Steam account. Keep Steam Guard enabled — the tool never stores your password, it uses Steam's standard authentication flow.
