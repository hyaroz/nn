Installation Guide

Setting up the Hitman Peacock Powershell script is designed to be as frictionless as possible. Follow the steps below to get your local server environment running in minutes.

Prerequisites

  • A legitimate, official installation of the Hitman trilogy purchased from a supported digital storefront.
  • Windows PowerShell (comes pre-installed with Windows 10/11).
  • Administrator privileges on your machine to map local network ports correctly.

Step-by-Step Setup

  • Step 1: Download the latest release archive from the project's GitHub repository.
  • Step 2: Extract the contents into a dedicated folder anywhere on your drive (e.g., C:\Peacock).
  • Step 3: Right-click on the Install-Peacock.ps1 script and select "Run with PowerShell".
  • Step 4: If prompted by User Account Control (UAC), click "Yes" to grant the necessary permissions for modifying the local host file.
  • Step 5: Wait for the script to automatically fetch the dependencies and configure the backend. Once the terminal displays "Server is listening", you are ready to launch the game.
