DepotDowngrader

DepotDowngrader is a lightweight wrapper around DepotDownloader that makes it quick and easy to fetch a specific historical build (depot/manifest) of a Steam application instead of whatever version is currently live. It's meant for cases where an older build behaves better for you than the current one — better performance, a favorite mod still being compatible, or avoiding a patch that broke something you relied on.

Project Overview

Under the hood, the tool talks to Steam's content servers the same way the official client does, just letting you pick a manifest ID for the exact version you want instead of always grabbing the latest one. It handles the manifest lookup, authentication with your Steam account, and file placement so you don't have to run everything by hand from the command line.

License requirement: DepotDowngrader only works for titles you already own on Steam. It authenticates through your own Steam account and downloads depots your account is entitled to — it does not provide, unlock, or grant access to any game or content you don't already have a valid license for.

Why Downgrade a Depot?

Developers sometimes ship updates that remove features, break mod compatibility, or introduce regressions. Rolling back to a previous build lets you keep playing the version that worked for you until things are sorted out upstream, without touching anything outside of the game's own files.

Compatibility Note

This tool is built and tested against official, unmodified Steam installations. It assumes a legitimate, licensed copy of the game — using it with modified or illegitimately obtained game files is outside of its intended use and isn't something this project supports.
