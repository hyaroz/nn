Installation Guide

Installing the FPS unlocker takes less than a minute. Make sure Dead by Daylight is fully updated through Steam or Epic before you begin.

Prerequisites

  • A legitimate installation of Dead by Daylight (Steam or Epic Games).
  • Windows 10 or 11.
  • Dead by Daylight fully closed — check the system tray to confirm the launcher isn't running in the background.

Step-by-Step Setup

  • Step 1: Download the latest release from the project's GitHub repository.
  • Step 2: Extract the archive. You should see DBD-FPS-Unlocker.exe and a Engine.ini patch file.
  • Step 3: Run DBD-FPS-Unlocker.exe as administrator. The tool detects your DBD install path automatically — if it can't find it, browse to the game folder manually (typically under Steam\steamapps\common\Dead by Daylight).
  • Step 4: Click Apply Patch. The unlocker backs up your original config and drops the override into the correct Engine\Config directory.
  • Step 5: Launch Dead by Daylight normally through Steam or Epic. Open the in-game settings and disable V-Sync if you want to see uncapped frames — then verify your FPS with the built-in counter or a tool like MSI Afterburner.

Reverting the Patch

Run the unlocker again and click Restore Original to put the default config back. Your backup is stored alongside the tool as Engine.ini.bak.

After a game update: Behaviour occasionally overwrites config files during patches. If the cap returns, re-run the unlocker and apply the patch again. Always grab the latest release that matches your current game version.
