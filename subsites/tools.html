Tools I Use

A curated list of software, utilities and apps I rely on daily. No fluff or filler — just the absolute best tools that earned a permanent spot on my machine.

OpenSteamTool

An open-source alternative to SteamTools based on custom .DLL files. It grants access to games and DLCs using .lua files and decryption keys, featuring a manifest downloader for seamless management.

View project arrow_outward

Squeegee Manifest App

A desktop app for browsing your library and easily installing Lua scripts or manifests. It uses the Hubcap API for seamless content management, making it a perfect companion for OpenSteamTool.

View project arrow_outward

MAS

A goated open-source script that saves you a ton of money by activating Windows and Office. It supports HWID, Ohook, TSforge, and Online KMS methods, complete with advanced troubleshooting tools.
